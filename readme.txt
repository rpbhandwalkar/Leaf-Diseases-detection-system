Web Project
Python 3 needs to be installed.
there is a requirements.txt file in the project
you can use pip install -r requirements.txt to install all the dependencies
run main.py to start the web application.

Andriod Application
open the project in android studio.
download the emulator or connect your phone via USB or wifi
Click on run
apk will be installed on your mobile and you can run the application
